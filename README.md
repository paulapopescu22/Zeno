## Project Overview

Zeno is a native iOS mobile application developed using Xcode and the Swift programming language. The application is designed to create a user-friendly and efficient environment by combining a dynamic search system with a booking management feature.

Its primary goal is to reduce time consumption for users by simplifying worker check-in processes during ongoing construction projects.

The application is specifically designed for construction companies operating in:

- Commercial buildings  
- Industrial facilities  
- Residential projects  
- Engineering infrastructure projects (roads, bridges, utility systems)  

Zeno allows companies to efficiently manage worker accommodation and booking processes while large-scale projects are in progress.
